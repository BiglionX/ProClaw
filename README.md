# 🦞 Proclaw Desktop

AI-Powered Business Operating System

## 快速开始

### 开发模式

```bash
npm run tauri dev
```

### 构建生产版本

```bash
npm run tauri build
```

## 技术栈

- **Framework**: Tauri 2.0 + React 18 + TypeScript
- **UI**: MUI + Tailwind CSS
- **State**: Zustand + React Query
- **Database**: SQLite (local) + Supabase (cloud)
- **AI**: Dify + Pinecone
