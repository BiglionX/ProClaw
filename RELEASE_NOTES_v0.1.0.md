## 新功能

### AI 经营团队
- 内置 **AI 经营团队**，包含 7 个专业 Agent：
  - 库存管理、销售预测、数据分析、采购管理
  - 财务管理、客户服务、AI 智能找图
- 团队创建/编辑/删除/导入/导出
- 团队发布到市场

### 产品库管理
- 简单模式 + SPU-SKU 电商模式
- 产品分类与品牌管理

### 进销存 AI 模块
- 供应商管理
- 采购订单 CRUD
- 库存管理与交易记录

### 仪表盘
- 实时数据图表
- 销售趋势分析
- 库存预警

### 技能商店
- 技能浏览与安装
- 技能市场基础架构

### 云托管商城 (预览)
- AI 生成独立站
- 主题/产品/订单/优惠券管理

## 技术栈

- **桌面框架**: Tauri 2.0 (Rust + React 18)
- **前端**: React 18 + TypeScript + MUI + Tailwind CSS
- **后端**: Rust (Axum) + SQLite (SQLCipher)
- **云端**: Supabase (可选)
- **测试**: Vitest + Playwright + Rust #[test]

## 安装说明

1. 下载 `ProClaw_0.1.0_x64-setup.exe`
2. 双击运行安装程序
3. 按向导完成安装
4. 启动 ProClaw Desktop

首次启动会自动初始化本地数据库，无需额外配置。
