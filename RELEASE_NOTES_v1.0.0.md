## 新功能

### 🧪 测试用户数据包（演示账号）

`boss / IamBigBoss` 登录后自动预置以下数据：

| 数据项 | 数量 | 详情 |
|--------|------|------|
| 产品 SPU | 20 | iPhone 电池，覆盖主流型号与容量 |
| 云商城 | 1 | `proclaw.cc/demo`，含 8 个管理 Tab + 预览 |
| AI 团队 | 3 | AI 经营 / 国内社媒 / 海外社媒（从 Nvwax 下载，失败回退本地映射） |
| 行业插件 | 1 | 外贸柜台运营助手（多语言 / 国际物流 / 海关申报） |

- 首次登录自动注入，欢迎提示列出全部预置内容
- TopBar 显示「🧪 演示数据」徽章，点击可跳转到设置
- 「设置 → 🧪 数据管理」Tab 可一键重置回出厂状态（含重置次数记录）
- 通过 Nvwax 接入失败时自动回退 `localTeamSkillMap`，保证离线可用

### 🎯 双模式架构正式上线

- **ProClaw Plus（进销存版）**：完整供应链管理 + AI团队 + 财务管理
  - 供应链：库存管理、采购管理、销售管理
  - 财务：损益报表、现金流、应收应付对账
  - AI团队：7个专业Agent协同工作

- **ProClaw Light（服务行业版）**：AI知识库 + 轻量级管理
  - AI知识库：三库合一（媒体库 + 问答库 + 资料库）
  - 简化的导航和页面布局
  - 适合单一门店、个体商户

### 🤖 AI 经营团队增强

- **CEO Agent 主控官系统**（PRD v6.2/v6.3）
  - 项目上下文协议（PCP）：愿景/目标/约束/里程碑
  - 任务分派与跟踪：自动分配任务给子Agent
  - 决策确认机制：审阅/驳回决策日志
  - 个性化偏好学习

- **Agent 生态**：25+ 行业Agent包
  - CRM、文档生成、HR、任务管理
  - 内容生成、转化优化、SEO
  - 社交媒体运营（中国/美国/东南亚）

### 🏭 行业插件系统（Phase 4）

- **餐饮行业**：POS收银、桌台管理、KDS厨房显示
- **美业行业**：预约管理、服务项目、员工管理、营销活动
- **宠物行业**：宠物档案、寄养管理、美容服务
- **Cloud版**：Token计费、债务保护、云端备份
- **会员管理**：跨行业通用会员体系

### ☁️ 云托管商城增强

- AI生成独立电商站点（Next.js 16 + React 19）
- 多套餐订阅（免费/基础/专业/企业）
- 商品同步、订单管理、优惠券管理
- 独立子域名 + 自定义域名

### 🔐 安全与性能

- 本地数据加密（SQLCipher AES-256-GCM）
- 邀请码防伪（HMAC-SHA256）
- 插件签名验证（Ed25519）
- Token用量监控和成本控制

## 技术栈

| 层级 | 技术 |
|------|------|
| **桌面框架** | Tauri 2.11 (Rust) |
| **前端** | React 18 + TypeScript + Vite 5 |
| **UI** | MUI 5 + Tailwind CSS 3 + Framer Motion |
| **状态管理** | Zustand + TanStack Query |
| **数据层** | SQLite + SQLCipher (本地) + Supabase (云端) |
| **HTTP 服务** | Axum 0.7 + WebSocket |
| **AI 框架** | LangChain + Ollama + 多LLM支持 |
| **加密** | AES-256-GCM + Argon2 + HMAC-SHA256 + Ed25519 |
| **图表** | Recharts + simple-statistics |
| **测试** | Vitest + Playwright + Rust #[test] |

## 安装说明

1. 下载 `ProClaw_1.0.0_x64-setup.exe`
2. 双击运行安装程序
3. 按向导完成安装
4. 启动 ProClaw Desktop

**快速体验**：使用内置模拟账号登录
- 用户名：`boss`
- 密码：`IamBigBoss`

> 🧪 演示账号 `boss / IamBigBoss` 默认已预置完整测试数据包（20 个产品 + 1 个云商城 + 3 个 AI 团队 + 外贸柜台插件），登录后即可在「设置 → 数据管理」查看与重置。
