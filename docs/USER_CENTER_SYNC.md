# 用户中心功能同步说明

## 📋 概述

已将营销网站的用户中心功能同步到桌面端应用的设置页面中，提供统一的用户体验。

## ✅ 已完成的功能

### 1. 新建文件

#### `src/pages/UserCenterPage.tsx`
完整的用户中心页面，包含：

**用户信息卡片**
- 用户头像（AccountCircle 图标）
- 用户名/邮箱显示
- 注册时间
- 退出登录按钮

**统计卡片（4个）**
- 💰 Token 余额
- 📊 总调用次数
- 🔑 活跃密钥数量
- 💵 累计消费金额

**功能 Tab（5个）**
1. **API 密钥管理** - 管理和配置 AI API 密钥
2. **Token 管理** - 查看余额、购买记录和充值
3. **使用统计** - API 调用统计和分析
4. **外部集成** - Webhook 和 API 端点配置
5. **账户设置** - 个人资料和安全设置

### 2. 修改文件

#### `src/pages/SettingsPage.tsx`
- ✅ 添加"👤 用户中心"作为第一个 Tab
- ✅ 导入 UserCenterPage 组件
- ✅ 调整原有 Tab 索引（AI模型→1, 数据库→2, 系统信息→3）

## 🎨 设计特点

### 与营销网站保持一致
- 相同的统计数据结构
- 相似的功能模块划分
- 统一的黑白灰配色方案
- Material-UI 组件库（桌面端）vs Tailwind CSS（营销网站）

### 桌面端优化
- 使用 Material-UI 组件，符合桌面应用风格
- Tab 面板布局，便于切换不同功能
- 响应式 Grid 布局，适配不同窗口大小
- 图标 + 文字的清晰导航

### 路由处理
- 当前作为 SettingsPage 的一个 Tab
- 退出登录后跳转到 `/login`

## 📍 访问路径

在桌面应用中：
1. 打开应用
2. 点击左侧导航栏的"⚙️ 设置"
3. 选择"👤 用户中心" Tab

## 🔄 与营销网站的差异

| 特性 | 营销网站 | 桌面端 |
|------|---------|--------|

| Profile 支持 | ✅ 有 profile 字段 | ❌ 仅 user 字段 |

1. **当前为演示模式**：所有统计数据都是硬编码的示例数据
2. **功能占位符**：各 Tab 内容显示"即将推出"提示
3. **退出登录**：会跳转到登录页面，需要重新登录
4. **无 Profile 支持**：桌面端 authStore 暂无 profile 字段，如需添加可参考营销网站实现
