## 🎨 设计系统

### 配色方案
- **蓝色系**: 用户、信息、主要操作
- **绿色系**: 成功、收入、活跃状态
- **紫色系**: API、分析、神秘感
- **橙色系**: 警告、通知、重要提示
- **红色系**: 错误、危险、删除操作
- **灰色系**: 中性、次要、禁用状态

### 组件规范
- **卡片**: 圆角xl，阴影sm，白色背景
- **按钮**: 圆角lg，统一的hover效果
- **徽章**: 圆角full，小号字体
- **表格**: 分隔线，hover高亮
- **表单**: 统一的输入框样式
- **模态框**: 圆角xl，最大宽度限制

### 响应式断点
- **手机**: < 768px (单列布局)
- **平板**: 768px - 1024px (双列布局)
- **桌面**: > 1024px (多列布局)
