# ProClaw v1.2 P1 - 批量导入中心 实施计划

> **状态**：实施中｜**目标**：商品/库存/采购/销售/供应商/客户 6 类业务对象批量导入

## 一、功能范围

### 1.1 支持的导入目标（6 类）
| 目标类型       | target_type | 主表                          | 关键字段                                  |
| -------------- | ----------- | ----------------------------- | ----------------------------------------- |
| 商品库         | products    | products                      | sku, name, category, brand, cost_price... |
| 库存交易       | inventory   | inventory_transactions        | sku, transaction_type, quantity, date     |
| 采购订单       | purchases   | purchase_orders + items       | po_number, supplier, order_date, sku...   |
| 销售订单       | sales       | sales_orders + items          | so_number, customer, order_date, sku...   |
| 供应商         | suppliers   | suppliers                     | name, code, contact_person, phone         |
| 客户           | customers   | customers                     | name, code, contact_person, customer_type |

### 1.2 支持的文件格式
- **CSV**（完整实现，UTF-8 BOM 剥离，csv 1.3 库）
- **JSON**（数组对象格式）
- **XLSX**（stub，提示用户「另存为 CSV」——安装包体积敏感）

### 1.3 7 步导入向导
1. **选目标 + 选文件**：选 6 类之一 + 上传文件
2. **解析预览**：表头 + 前 3 行样例
3. **字段映射**：自动按中英别名匹配，可手动调整
4. **校验**：必填/类型/日期/枚举校验，错误清单
5. **开始导入**：事务化执行，成功/失败计数
6. **结果报告**：成功率 + 错误下载
7. **完成**：可继续导入 / 返回中心

## 三、后端模块（src-tauri/src/import/）

```
src-tauri/src/import/
├── mod.rs          // 模块声明 + 公共 re-export
├── types.rs        // ImportTarget / BatchStatus / FieldDef / 6 类字段常量
├── parser.rs       // parse_csv / parse_json / parse_xlsx_stub
├── executor.rs     // validate_batch / execute_batch / 6 个 process_*
├── templates.rs    // 6 套 CSV 模板 + ensure_templates
└── commands.rs     // 13 个 Tauri 命令
```

### 3.1 13 个 Tauri 命令

| 命令 | 用途 |
| --- | --- |
| `import_create_batch` | 上传文件 + 创建批次 + 解析 |
| `import_parse_file` | 重新解析已上传文件 |
| `import_upload_file` | deprecated（保留兼容） |
| `import_validate_batch` | 字段映射后校验 |
| `import_execute_batch` | 事务化执行 |
| `import_pause_batch` | 暂停 |
| `import_cancel_batch` | 取消 |
| `import_retry_batch` | 重新执行 |
| `import_get_batch` | 查询单批次 |
| `import_list_batches` | 历史批次列表 |
| `import_get_batch_errors` | 错误日志 |
| `import_get_templates` | 6 套模板元数据 |
| `import_list_mapping_templates` | 用户保存的映射模板 |
| `import_save_mapping_template` | 保存映射模板（upsert） |

### 3.2 幂等策略
- **products**：`sku` upsert
- **inventory**：(product_id, date, transaction_type, reference_no) 跳过重复
- **purchases**：`po_number` upsert（items 全量替换）
- **sales**：`so_number` upsert（items 全量替换）
- **suppliers**：先 `code` 后 `name` upsert
- **customers**：先 `code` 后 `name` upsert

### 3.3 启动时模板拷贝
- 路径：`APPDATA/proclaw/desktop/import_templates/`
- 启动时 `ensure_templates()` 一次性写入 6 套 CSV
- 用户从导入中心下载

| --- | --- | --- |
