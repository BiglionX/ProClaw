## 1. 背景与现状诊断

营销网站（`marketing-site/`，Vite + React + MUI + Tailwind）当前承担品牌落地、用户获取、产品介绍、用户中心等核心职责。经全面审查，存在以下问题：

### 1.1 内容过时

| 问题 | 位置 | 影响 |
|------|------|------|
| 技术栈标注"Tauri 2.0"，实际已是 2.11 | HomePage.tsx L238 | 技术形象陈旧 |
| 版本号标注 v0.1.0，实际已 v1.0.0-beta.2 | HomePage.tsx L151 | 版本认知滞后 |
| 未提及 Light 极简版 | 全局 | 丢失重要产品线曝光 |
| 未展示行业插件生态（餐饮/美业/宠物/Cloud） | HomePage / FeaturesPage | 最大卖点缺失 |
| SEO 描述仅提及"7 个 Agent"，实际已有 25+ Bundles | index.html L7 | 搜索曝光不准确 |

### 1.2 页面缺口

| 缺口 | 严重程度 | 说明 |
|------|----------|------|
| **无行业方案页** | 🔴 高 | Phase 4 行业插件已完成，营销站没有任何行业落地页 |
| Navbar 缺少 `/plugins` 入口 | 🟡 中 | PluginStorePage 已开发但导航无入口 |
| 下载页无移动端 APK | 🟡 中 | mobile/ 已开发但下载页仅展示桌面端 |
| 无用户案例/社会证明 | 🟡 中 | 缺少信任建立的转化关键环节 |
| 无博客/资源中心 | 🟢 低 | 影响 SEO 长尾流量获取 |

### 1.3 技术债务

| 问题 | 详情 |
|------|------|
| UserCenterPage.tsx 51KB | 单文件过大，需拆分为子组件 |
| 组件复用率低 | Navbar/Footer 之外无共享 UI 组件库 |
| 无响应式图片优化 | 缺少 srcset、WebP 等现代图片策略 |

---

## 2. 优化目标

1. **品牌刷新** — 使营销站准确反映产品当前状态（三模式、行业插件、25+ Agent）
2. **转化提升** — 补齐行业方案页、社会证明等转化关键页面
3. **SEO 增强** — 为行业关键词（餐饮管理系统、美业预约软件等）建立落地页
4. **技术减债** — 拆分超大组件，提升可维护性

## 3. 功能需求

### 3.1 P0 — 主页内容刷新（HomePage 重构）

**当前问题**：技术栈版本过时、未展示行业插件、缺少 Light 版入口。

**改造内容**：

- 标语升级为：**"一个软件，所有行业——餐饮·零售·美业·宠物"**

#### 3.1.2 特性卡片区（featureCards 数组）

#### 3.1.3 新增"行业场景速览"区域


### 3.2 P0 — 新增行业方案页（`/solutions/:slug`）

**背景**：Phase 4 完成了餐饮/美业/宠物/Cloud 四个行业插件，但营销站没有任何展示。行业方案页是 SEO 长尾关键词的核心载体。

**页面结构**：

```
/solutions
  ├── /catering    → 餐饮行业方案
  ├── /beauty      → 美业行业方案
  ├── /pet         → 宠物行业方案
  └── /cloud       → Cloud 托管方案
```

#### 3.2.1 行业方案页模板

每个方案页包含以下板块：

| 板块 | 内容 |
|------|------|
| **Hero** | 行业场景图（CSS 插画或示意图）+ 一句话价值主张 |
| **痛点与解法** | 3 个典型痛点 → ProClaw 解法对比 |
| **功能亮点** | 该行业专属功能卡片（如餐饮：KDS/桌台/POS；美业：预约/排班/营销） |
| **AI 赋能** | 该行业专属 AI Agent 介绍（如餐饮：智能菜单优化、美业：客资分析） |
| **定价** | 基础桌面端免费 + 高级插件定价 |
| **CTA** | 免费下载 + 联系咨询 |

#### 3.2.2 各行业内容要点

**餐饮方案 (`/solutions/catering`)**：
- 功能：POS 收银、桌台管理、KDS 厨房显示、菜单管理
- AI Agent：`ma_catering_assistant` `ma_catering_kds` `ma_catering_menu`
- SEO 关键词：餐饮管理系统、餐厅POS软件、厨房显示系统

**美业方案 (`/solutions/beauty`)**：
- 功能：预约管理、服务项目、员工排班、营销活动
- AI Agent：`ma_beauty_assistant` `ma_beauty_scheduler` `ma_beauty_marketing`
- SEO 关键词：美业管理系统、美容院预约软件、美发店管理

**宠物方案 (`/solutions/pet`)**：
- 功能：宠物档案、寄养管理、美容预约
- AI Agent：`ma_pet_assistant` `ma_pet_boarding` `ma_pet_health`
- SEO 关键词：宠物店管理系统、宠物寄养软件、宠物美容预约

**Cloud 方案 (`/solutions/cloud`)**：
- 功能：Token 计费、云端备份、云商城托管
- AI Agent：`ma_cloud_backup` `ma_cloud_billing` `ma_cloud_ops`
- SEO 关键词：云进销存、Token计费、数据云备份
