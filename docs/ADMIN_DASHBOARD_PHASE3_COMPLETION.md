## 💡 使用场景

### 1. 自定义报表
**场景**: 运营经理需要每周生成平台运营报告
- 选择"用户增长报表"模板
- 设置时间范围为"近7天"
- 选择"按天"分组
- 点击"生成报表"
- 预览数据和图表
- 导出为CSV发送给团队

### 2. 自动化任务
**场景**: 系统管理员监控定时任务执行情况
- 查看所有任务列表
- 发现"同步外部数据"任务失败
- 点击"查看日志"排查问题
- 修复后点击"立即执行"测试
- 确认成功后恢复正常运行

### 3. API限流
**场景**: 防止API滥用和保护系统稳定
- 创建新的端点限流规则
- 设置`/api/chat`每分钟20次请求
- 配置突发容量为30
- 添加VIP用户到白名单
- 启用规则并监控效果

### 4. 审计日志
**场景**: 安全团队调查可疑操作
- 搜索特定用户的操作记录
- 筛选"失败"状态的操作
- 查看最近24小时的登录尝试
- 点击详情查看完整信息
- 导出日志作为证据保存
