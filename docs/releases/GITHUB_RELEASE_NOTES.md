# 🦞 ProClaw Desktop v0.1.0-beta 发布说明

## 📦 发布信息

- **版本号**: v0.1.0-beta
- **发布日期**: 2026-04-12
- **平台**: Windows 10/11 (64位)
- **安装包**: `Proclaw_0.1.0_x64_en-US.msi`
- **大小**: ~4.2 MB
- **许可证**: MIT

## ✨ 新功能特性

ProClaw Desktop 是为中小企业设计的**一体化商业管理平台**，主要功能包括：

### 🎯 核心功能

#### 1. 产品库管理

- ✅ 完整的 CRUD 操作（创建、读取、更新、删除）
- ✅ 产品分类和品牌管理
- ✅ 图片上传和预览（Base64 存储）
- ✅ 高级筛选（按分类、品牌、关键词）
- ✅ CSV 数据导出
- ✅ 后端 SQL 级过滤（高性能）

#### 2. 进销存管理（升级版）

- ✅ 库存交易记录（入库/出库/调整/调拨）
- ✅ 实时库存统计仪表板
- ✅ 低库存预警系统
- ✅ 库存充足性验证
- ✅ 交易历史查询
- ✅ **新增**: AI 驱动的库存优化

#### 3. AI 经营智能体

- ✅ 自然语言指令解析（规则匹配）
- ✅ 智能对话界面
- ✅ 快捷操作面板
- ✅ 跨页面浮动助手

#### 4. 系统架构

- ✅ Tauri 2.0 桌面框架（轻量级）
- ✅ SQLite + SQLCipher 本地加密数据库
- ✅ Supabase 云端同步（可选）
- ✅ 离线优先架构
- ✅ 跨平台支持（Windows/macOS/Linux）

#### 5. 用户体验

- ✅ 演示模式（无需配置即可使用）
- ✅ 演示账号快速登录
- ✅ 响应式 UI 设计
- ✅ 深色/浅色主题支持

## 📥 下载安装

### Windows 安装

1. **下载** MSI 安装包：`Proclaw_0.1.0_x64_en-US.msi`
2. **双击** 安装包文件
3. **按照** 安装向导完成安装
4. **启动** ProClaw Desktop

### 快速开始

#### 方式 1: 演示模式（推荐新手）

1. 安装并启动 ProClaw Desktop
2. 点击"进入演示模式"
3. 使用演示账号登录：
   - 用户名: `boss`
   - 密码: `IamBigBoss`
4. 立即开始体验！

#### 方式 2: 云端模式（生产环境）

1. 注册免费的 [Supabase](https://supabase.com) 账号
2. 创建新项目，获取 API Key
3. 在 ProClaw 设置页面配置 Supabase
4. 使用邮箱注册/登录
5. 享受云端同步和多设备访问
