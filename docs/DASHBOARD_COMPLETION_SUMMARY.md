## 📈 数据流

```
用户访问仪表盘
    ↓
组件挂载 (useEffect)
    ↓
loadDashboardData()
    ↓
Promise.all([
  getInventoryStats(),
  getSalesTrend('day'),
  getProductAnalytics(),
  getFinancialSummary(),
  getDatabaseStats()
])
    ↓
更新 State
    ↓
React 重新渲染
    ↓
显示数据和图表
```
