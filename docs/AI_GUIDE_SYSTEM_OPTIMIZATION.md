# AI智能引导系统优化总结

## 📋 优化概览

本次优化全面升级了ProClaw的AI驱动决策支持系统，添加了智能检测、个性化推荐、错误处理和功能发现等核心功能。

---

## ✨ 新增功能

### 1. **LLM连接自动检测** 🔌
- **触发时机**: 用户首次打开AI Chat时
- **检测内容**: 
  - 检查是否配置了任何活跃的LLM提供商
  - 识别默认提供商（ProClaw集成LLM）
  - 显示当前连接的提供商列表
- **引导策略**:
  - 未连接: 显示详细的配置指南和支持的服务商列表
  - 已连接: 显示连接状态并展示个性化推荐

### 2. **智能意图识别** 🧠
支持10种意图类型：
- `llm_connection` - LLM连接相关问题
- `deployment` - 部署和安装问题
- `pricing` - 费用和定价问题
- `business_query` - 业务数据查询
- `setup_guide` - 设置和使用指南
- `error_troubleshoot` - 错误排查
- `feature_discovery` - 功能发现
- `data_operation` - 数据导入/导出
- `general_chat` - 一般对话
- `unknown` - 未知意图

**识别机制**:
- 基于关键词匹配的评分系统
- 置信度计算（0-1范围）
- 动态建议生成

### 3. **错误处理与故障排除** 🔧

#### 支持的错误类型：
1. **API密钥问题** (`api_key_invalid`)
   - 密钥格式错误
   - 密钥过期或撤销
   - 服务未开通
   
2. **网络连接问题** (`connection_failed`)
   - 网络不稳定
   - 防火墙/代理阻止
   - API地址配置错误
   
3. **模型不存在** (`model_not_found`)
   - 模型名称拼写错误
   - 账户未开通该模型
   - 模型已下线
   
4. **权限不足** (`permission_denied`)
   - API密钥权限不足
   - 账户等级限制
   - 组织权限设置
   
5. **超时错误** (`timeout_error`)
   - 网络延迟过高
   - 服务器负载高
   - 请求数据量过大

#### 每个错误提供：
- 可能的原因分析
- 详细的解决步骤
- 相关的支持信息

### 4. **功能发现与推荐** 🎯

#### 智能推荐场景：
1. **库存预警**
   - 触发词: 库存、预警、提醒
   - 推荐: 智能库存预警功能
   - 路径: `/inventory`

2. **销售数据分析**
   - 触发词: 销售、分析、趋势、报表
   - 推荐: AI驱动的销售预测
   - 路径: `/analytics`

3. **智能采购建议**
   - 触发词: 采购、供应商、订单
   - 推荐: 基于预测的采购建议
   - 路径: `/purchase`

4. **批量数据导入**
   - 触发词: 导入、批量、excel
   - 推荐: Excel/CSV批量导入
   - 路径: `/products`

### 5. **数据操作指南** 📊

#### 数据导入支持：
- Excel/CSV格式
- 批量产品、库存、客户、供应商导入
- 自动数据验证
- API批量导入
- 从其他ERP系统迁移

#### 数据导出支持：
- 产品列表（含图片、价格、库存）
- 库存交易记录
- 销售订单和记录
- 客户和供应商信息
- 财务报表
- 支持Excel、CSV、JSON格式
- 定时自动备份

### 6. **个性化推荐** 💡

#### 基于用户行为分析：
1. **查询历史分析**
   - 记录最近20条查询
   - 识别查询模式
   - 推荐相关功能

2. **功能使用统计**
   - 追踪各页面访问频率
   - 识别常用功能
   - 推荐进阶用法

3. **示例推荐**:
   - 查询过"库存" → 推荐库存预警设置
   - 查询过"销售" → 推荐销售预测功能
   - 频繁访问产品页 → 推荐分类管理

### 7. **用户行为记录** 📈

#### 记录内容：
```typescript
interface UserBehavior {
  queryHistory: string[];      // 查询历史
  lastActivePage: string;       // 最后活跃页面
  featureUsage: Record<string, number>; // 功能使用次数
}
```

#### 存储策略：
- 本地存储（localStorage）
- 保留最近20条查询记录
- 隐私保护（仅本地使用）

## 🏗️ 技术架构

### 核心模块

#### 1. `aiGuide.ts` - 智能引导引擎
```typescript
// 主要函数
- checkLLMConnectionStatus()     // 检查连接状态
- detectIntent()                 // 意图识别
- handleUserInput()              // 处理用户输入
- getConnectionGuideMessage()    // 获取连接引导
- troubleshootError()            // 错误排查
- discoverFeatures()             // 功能发现
- showDataOperations()           // 数据操作指南
- getPersonalizedRecommendations() // 个性化推荐
- recordUserBehavior()           // 记录用户行为
```

#### 2. `FloatingAgentChat.tsx` - 聊天界面集成
- 自动连接检测
- 个性化推荐展示
- 用户行为记录
- 智能路由（引导类 vs 业务类）

### 工作流程

```
用户输入
  ↓
意图识别 (detectIntent)
  ↓
检查LLM连接状态
  ↓
┌─────────────────────┐
│  是否有引导响应？    │
└─────────────────────┘
  ↓ Yes          ↓ No
返回引导    执行commandParser
  ↓
显示响应
  ↓
记录用户行为
```
