## 🐛 常见问题排查

### 问题1: 页面显示空白

**可能原因**:
- 路由配置错误
- 组件导入失败

**解决方法**:
```bash
# 检查控制台错误
# 确认文件路径正确
# 重新安装依赖
npm install
```

### 问题2: 通知铃铛不显示

**可能原因**:
- NotificationProvider 未正确包裹
- 导入路径错误

**解决方法**:
- 检查 `App.tsx` 中是否有 `<NotificationProvider>`
- 确认导入语句：`import { NotificationBell } from '../lib/notificationContext'`

### 问题3: 导出文件乱码

**可能原因**:
- CSV BOM 未正确添加
- 文件编码问题

**解决方法**:
- 检查 `exportUtils.tsx` 中的 `\uFEFF` BOM 标记
- 用支持UTF-8的编辑器打开

### 问题4: 统计数据不显示

**可能原因**:
- 模拟数据加载失败
- useState 初始化问题

**解决方法**:
- 检查浏览器控制台错误
- 确认 `setTimeout` 模拟延迟正常工作
