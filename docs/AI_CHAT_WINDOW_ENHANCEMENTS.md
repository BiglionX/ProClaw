# AI Chat 窗口增强功能

## 📋 功能概述

为 ProClaw 的 AI Chat 浮动窗口添加了以下增强功能，提升用户体验和操作便利性：

### ✨ 新增功能

1. **🖱️ 拖拽调整窗口大小**
   - 在窗口顶部边缘（标题栏上方）有一个隐藏的拖拽区域
   - 鼠标悬停时会显示红色高亮提示
   - 按住鼠标左键向上/向下拖动可以调整窗口高度
   - 按住鼠标左键向左/向右拖动可以调整窗口宽度
   - 最小尺寸：350px × 300px
   - 最大尺寸：受屏幕尺寸限制

2. **🔲 最大化/恢复窗口**
   - 点击标题栏右侧的"最大化"按钮（□ 图标）
   - 窗口将扩展到几乎全屏（保留 48px 边距）
   - 再次点击"恢复"按钮（⧉ 图标）可恢复到之前的大小
   - 最大化状态下禁用拖拽调整功能

3. **💾 自动记忆窗口尺寸**
   - 系统会自动保存您调整的窗口尺寸到浏览器本地存储
   - 下次打开 AI Chat 时，会自动使用上次的大小
   - 最小化和最大化状态不会被保存

4. **🎯 优化的交互体验**
   - 拖拽时禁用页面文本选择，避免误操作
   - 拖拽时光标变为调整大小图标（↕️）
   - 平滑的过渡动画（拖拽时禁用动画以提升性能）
   - 所有按钮都有 Tooltip 提示

## 🎮 使用方法

### 调整窗口高度（上拉展开）

1. 将鼠标移动到窗口顶部边缘（标题栏上方约 8px 的区域）
2. 当光标变成上下箭头（↕️）时，表示可以拖拽
3. 按住鼠标左键向上拖动 → 窗口变高
4. 按住鼠标左键向下拖动 → 窗口变矮
5. 释放鼠标完成调整

### 调整窗口宽度

1. 同样在窗口顶部边缘拖拽
2. 向左拖动 → 窗口变宽
3. 向右拖动 → 窗口变窄

### 最大化窗口

1. 点击标题栏右侧的"最大化"按钮（□ 图标）
2. 窗口会立即扩展到接近全屏
3. 适合查看长对话历史或需要更大显示区域时

### 恢复窗口大小

1. 在最大化状态下，点击"恢复"按钮（⧉ 图标）
2. 窗口会恢复到最大化之前的大小

### 最小化窗口

1. 点击标题栏右侧的"最小化"按钮（— 图标）
2. 窗口会缩小为一个圆形图标
3. 点击圆形图标可重新展开

## 💡 使用场景

### 场景 1：查看长对话历史
- **问题**：对话内容很多，需要滚动查找之前的消息
- **解决**：使用最大化功能，获得更大的显示区域，减少滚动次数

### 场景 2：多任务处理
- **问题**：需要在 AI Chat 和其他应用之间切换
- **解决**：调整窗口到合适的大小，既不影响其他工作，又能快速查看 AI 回复

### 场景 3：个性化偏好
- **问题**：不同用户喜欢的窗口大小不同
- **解决**：系统自动记忆您的偏好，每次打开都是您喜欢的大小

## 🔧 技术实现

### 核心特性

- **响应式拖拽**：使用 `mousemove` 和 `mouseup` 事件监听器
- **尺寸限制**：防止窗口过小或过大影响使用
- **本地存储**：使用 `localStorage` 持久化用户偏好
- **性能优化**：拖拽时禁用 CSS 过渡动画，提升流畅度
- **状态管理**：正确处理最小化、最大化、正常状态的切换

## 🎨 UI 设计

### 标题栏按钮布局（从左到右）

1. **最大化/恢复按钮** - □ / ⧉
2. **最小化按钮** - —
3. **关闭按钮** - ✕

### 拖拽区域

- 位置：窗口顶部边缘（标题栏上方）
- 高度：8px
- 悬停效果：淡红色背景提示
- 光标：上下箭头（ns-resize）

## 📝 注意事项

1. **最小化状态下**：无法进行拖拽调整和最大化操作
2. **最大化状态下**：拖拽功能被禁用，需要先恢复窗口
3. **尺寸限制**：
   - 最小高度：300px（保证基本可用性）
   - 最小宽度：350px（保证内容可读性）
   - 最大高度：屏幕高度 - 150px
   - 最大宽度：屏幕宽度 - 100px

4. **数据持久化**：
   - 窗口尺寸保存在浏览器的 `localStorage` 中
   - 清除浏览器数据会重置为默认尺寸
   - 不同浏览器/设备需要分别设置
