## AI团队页面架构

### 页面布局

| 区域 | 功能 |
|------|------|
| 侧边栏 | Agent列表选择 |
| 主区域 | 聊天交互区域 |
| 底部 | 输入框 |

### Agent类型

| Agent名称 | 功能定位 | 上下文配置 |
|-----------|----------|------------|
| CEO Agent | 全局决策、战略建议 | 系统级上下文 |
| 财务Agent | 财务对账、报表生成 | 财务数据上下文 |
| 秘书Agent | 日程管理、提醒服务 | 个人信息上下文 |
| 客服Agent | 客户咨询、问题解答 | 客户数据上下文 |

## 交互流程

### 1. Agent选择
- 用户从侧边栏选择目标Agent
- 切换Agent时保持对话历史
- 当前Agent高亮显示

### 2. 对话交互
- 消息发送：支持文本、语音输入
- 消息展示：流式输出、Markdown渲染
- 状态指示：输入中、思考中、完成

### 3. Agent协作
- 跨Agent任务转发
- 上下文共享机制
- 结果汇总展示

## 技术实现

### 组件结构
- AgentSidebar.vue：侧边栏组件
- AgentChat.vue：聊天组件
- AgentMessage.vue：消息气泡
- AgentInput.vue：输入框组件
