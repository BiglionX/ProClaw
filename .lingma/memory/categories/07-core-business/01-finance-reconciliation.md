# 财务对账与应收应付管理

> 创建时间：2026-06-08  
> 最后更新：2026-06-08  
> 关键词：对账功能、应收应付、自动提醒、邮件发送、财务管理

## 对账功能需求

### 核心功能

| 功能模块 | 描述 | 优先级 |
|----------|------|--------|
| 对账单生成 | 自动生成客户/供应商对账单 | P0 |
| 自动提醒 | 到期付款自动提醒 | P0 |
| 邮件发送 | 对账单邮件通知 | P0 |
| 应收管理 | 应收账款跟踪 | P1 |
| 应付管理 | 应付账款管理 | P1 |
| 账龄分析 | 应收账款账龄分析 | P2 |

### 对账流程

1. 数据采集：从订单系统获取交易数据
2. 数据匹配：自动匹配收付款记录
3. 差异处理：人工处理无法自动匹配的差异
4. 对账确认：双方确认对账结果
5. 报表生成：生成对账报表

### 自动提醒机制

- 付款到期前3天首次提醒
- 付款当天二次提醒
- 逾期后每天提醒
- 邮件+站内信双通道

## 应收应付管理

### 应收款流程
```
销售订单 → 发票开具 → 应收登记 → 收款确认 → 核销
```

### 应付款流程
```
采购订单 → 收货确认 → 应付登记 → 付款审批 → 付款执行
```

## 数据模型

| 表名 | 用途 |
|------|------|
| reconciliation_bills | 对账单主表 |

| ar_records | 应收记录 |
