## 项目概述

ProClaw 是一款面向中小企业的智能化管理系统，采用插件化架构，支持多行业解决方案。

## 技术栈

- **前端**：React + TypeScript + Vite
- **桌面端**：Tauri (Rust)
- **移动端**：React Native (Expo)
- **云端**：Next.js + Supabase
- **AI**：DeepSeek API 集成

## 核心功能模块

### 1. 身份管理
- 小如（AI助手）作为核心身份管理入口
- 对话式交互规范已完成定义

### 2. AI智能体体系
- CEO Agent：全局决策助手
- AI客服：独立上下文配置
- AI团队：多Agent协作页面

### 3. 财务管理
- 对账功能：自动提醒 + 邮件发送
- 应收应付管理
- 对账单生成

### 4. 插件系统
- 行业插件：餐饮、美容、宠物等
- 云插件：云端扩展能力
- 插件签名与验证机制

## 产品版本

- **Light**：轻量版，桌面端
- **Cloud-Store**：云端版本
- **Mobile**：移动端应用
- **Plus**：增强版（包含AI能力）
