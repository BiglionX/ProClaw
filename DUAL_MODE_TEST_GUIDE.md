# 双模式商品库系统 - 功能测试指南

## 📋 测试前准备

1. 确保应用已启动：`npm run tauri dev`
2. 准备一些测试商品数据（至少3-5个商品）
3. 建议先备份数据库

## 🐛 已知问题和注意事项

1. **大数据量性能**：如果商品数量超过1000个，迁移可能需要较长时间，建议添加分批处理
2. **图片迁移**：目前只迁移image_url字段，如果有多个图片需要额外处理
3. **规格信息**：升级后所有SKU的规格都是"标准"，需要手动添加多规格
